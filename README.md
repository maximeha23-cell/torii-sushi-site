# Torii Sushi — site web

Site bilingue français/anglais du restaurant Torii Sushi à Laval. Il présente
le restaurant, ses créations, ses photos et ses coordonnées, avec accès direct
aux réservations Resy, aux menus et aux réseaux sociaux officiels.

## Prérequis

- Node.js `>=22.13.0`

## Développement local

```bash
npm install
npm run dev
npm run build
```

## Pages

- `/` — accueil en français
- `/a-propos` — à propos
- `/photos` — galerie
- `/en` — English home
- `/en/about-us` — about
- `/en/photos` — gallery

## Déploiement

Le projet est préparé pour OpenAI Sites avec la configuration située dans
`.openai/hosting.json`.
